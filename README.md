# Apalpador

Adds the traditional Galician Christmas character "Apalpador" to your WordPress site with festive visual effects.

## Features

- Customizable Apalpador image and position
- Entry, idle, and click animations
- Snow effect with configurable density
- Shooting star effect
- Respects user accessibility preferences (prefers-reduced-motion)
- Compatible with WooCommerce (disabled on cart/checkout)

## Installation

1. Upload the `apalpador` folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Apalpador' in the admin menu to configure settings
