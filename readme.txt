Adds the traditional Galician Christmas character "Apalpador" to your WordPress site with festive visual effects.

== Description ==

The Apalpador is a traditional Galician Christmas figure, a charcoal maker who comes down from the mountains on the night of December 31st to check if children have eaten well during the year by touching their bellies.

This plugin places the Apalpador figure on your WordPress pages with festive visual effects including:

* Customizable Apalpador image and position
* Entry, idle, and click animations
* Snow effect with configurable density
* Shooting star effect

= Features =

* Easy to configure through WordPress admin
* Date range settings to show only during the holiday season
* Multiple animation options
* Performance optimized
* Respects user accessibility preferences (prefers-reduced-motion)
* Compatible with WooCommerce (disabled on cart/checkout)

== Installation ==

1. Upload the `apalpador` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Apalpador' in the admin menu to configure settings

== Frequently Asked Questions ==

= When does the Apalpador appear? =

By default, the Apalpador appears from December 1st to January 6th. You can customize these dates in the plugin settings.

= Can I use my own image? =

Yes! You can upload your own image through the WordPress media library in the plugin settings.

= Does it work with caching plugins? =

Yes, the plugin is designed to be compatible with caching systems.
